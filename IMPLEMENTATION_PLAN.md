# 究極の双方向リモート操作ハブ (OmniControl Hub) 実装計画

スマホからPCを、PCからスマホを相互に制御し、あらゆる便利機能を統合した「究極のハブ」を構築します。

## 1. 相互操作・コア機能
### A. スマホ → PC (リモートコントローラー)
- **プレシジョン・トラックパッド**: 慣性スクロール、マルチタッチ（右クリック、スクロール）対応。
- **フルリモートキーボード**: スマホのソフトキーボードからPCへ入力。
- **ショートカット・マクロ**: 1タップで「コピー」「ペースト」「ウィンドウ閉じ」などを実行。
- **システム制御**: 音量、輝度、メディア再生、スリープ/シャットダウン。
- **アプリランチャー**: PC上の特定のアプリをスマホから起動。

### B. PC → スマホ (デバイスボイス/通知)
- **PCからメッセージ送信**: PCで入力したテキストをスマホのクリップボードへ即座に送信。
- **URL送出**: PCで見ているページをスマホで即座に開く。
- **通知転送**: PCからスマホへ通知を飛ばす（WebPush/Socket経由）。

### C. 共有・連携 (便利機能)
- **ユニバーサル・クリップボード**: PCでコピーしたテキストがスマホでペースト可能に（逆も然り）。
- **ドラッグ＆ドロップ・ファイル転送**: スマホからPC、PCからスマホへファイルを無線転送。
- **スクリーンキャスト**: PCの画面をスマホで低遅延プレビュー。
- **リモート・ショートカット**: PCのショートカットキー（Alt+Tabなど）をスマホから操作。

## 2. 技術スタック
- **Frontend/Backend**: Next.js (App Router) + Tailwind CSS (UI用/ユーザー要求があれば) -> 今回は高級感を出すため **Vanilla CSS + Glassmorphism** を採用。
- **Real-time**: Socket.io
- **PC Automation**: `@nut-tree/nut-js` (マウス・キーボード操作)
- **System Metrics**: `node-os-utils`
- **Utility**: `qrcode` (接続用), `busboy` (ファイル転送)

## 3. UI/UX デザインコンセプト
- **Neon-Glassmorphism**: 半透明の背景、ネオン輝くグラデーション、滑らかなアニメーション。
- **Responsive Adaptive**: スマホではコントローラーUI、PCではダッシュボードUIに自動切替。

## 4. 実装フェーズ
1. **基盤構築**: Next.js + Socket.io Server のセットアップ。
2. **PC制御の実装**: `@nut-tree/nut-js` によるコマンド受付。
3. **双方向連携の構築**: クリップボード同期、ファイル転送、URL共有。
4. **プレミアムUIの実装**: PCダッシュボードとスマホリモコン画面の作成。
5. **接続性の向上**: QRコード表示、ネットワーク自動認識。
